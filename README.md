# CECYFLIX

Despliegue en la nube con Render

---

Este proyecto fue creado con [Create React App](https://github.com/facebook/create-react-app).

## Scripts disponibles

En el directorio del proyecto puedes ejecutar:

### `npm start`

Ejecuta la app en modo desarrollo.\
Abre [http://localhost:3000](http://localhost:3000) para verla en tu navegador.

La página se recargará si haces cambios.\
También puedes ver errores en la consola.

### `npm test`

Ejecuta pruebas en modo interactivo.

### `npm run build`

Genera la versión de producción en la carpeta `build`.

### `npm run eject`

⚠️ Este comando es irreversible. Copia las configuraciones y dependencias para personalizar el proyecto.
